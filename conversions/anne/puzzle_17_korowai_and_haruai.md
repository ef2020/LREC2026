# Puzzle 17 — Korowai and Haruai

Source: UKLO 2022 — https://www.uklo.org/wp-content/uploads/2022/08/7_UKLO-2022-Korowai-and-Haruai__Complete-Script.zip

## Context

Korowai is a Greater Awyu language spoken by about 3,500 people in Indonesia. Haruai is a Madang language spoken by about 2,000 in Papua New Guinea. Despite not being closely related, their numeral systems share many features -- both use body-count systems.

**Korowai body-count words (number):**
senan = little finger (1), senanafuel = ring finger (2), pinggup = middle finger (3), wayafuel = index finger (4), wayo = thumb (5), gedun = wrist (6), lafol = forearm (7), bonggup = elbow (8), labul = upper arm (9), main = shoulder (10), khomofekholol = neck (11), khotokhal = ear (12), khabean = head (13). Then 14-25 count back down using me(n)- prefix: 26-X form.

**Haruai body-count words (number):**
aglng = little finger (1, 19), agnyobo rolyobo = ring finger (2, 20), woloml = middle finger (3, 21), konyo ngb = index finger (4, 22), momd = thumb (5, 23), yngosw = bracelet (6, 18), mj = forearm (7, 17), amnyab = elbow (8, 16), mac = biceps (9, 15), ragpb = shoulder (10, 14), gadloy = collar bone (11, 13), mol = hole (12).

Attributive use: In Korowai, suffix -anop and place after the noun. In Haruai, no special marking.

## Notes
- Converted from Q 7.1 (translate into Korowai: little finger, 21, 4 index fingers), Q 7.2 (translate into Haruai: index finger, 5, 23 ring fingers), and Q 7.3 (translate into English: 5 items). All 11 items combined.

- This is a numeral system puzzle (tagged "Nu"), which works well as Match-Up because it pairs expressions with their numeric/body-part meanings.
