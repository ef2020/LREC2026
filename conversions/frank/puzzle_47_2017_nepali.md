# Puzzle 47 — Nepali

Source: UKLO 2017 — https://www.uklo.org/wp-content/uploads/2022/05/2017_R2.1-Nepali.pdf

## Context

Nepali is the official language of Nepal. It is written in Devanagari script, transliterated here with special marks (ī, ḍ, ț, etc.).

Additional vocabulary: ek bațțā churoț = one packet of cigarettes; das rupiyã = ten rupees; kāpī = notebook; nātini = granddaughter; mițhāī = sweets/candies; parivār = family; pēnsila = pencil; koțhaharu = rooms; śrīman = husband; sãco = key.

## Notes
- Context table (16 sentence pairs) and Match-Up table (10 translation items from Q1) combined into a single Match-Up table of 26 items.
- Some sentences accept two valid Nepali forms (e.g., -sita/-sãga variants). Only one form is shown in the table; either would be correct.
- The solver must deduce the possessive particle system: -sita/-sãga (alienable possession), -ko/-ki/-ka (inalienable/relational), question (cha?), negation (chaina), and plural (chan with numeral).
