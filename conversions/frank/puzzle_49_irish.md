# Puzzle 49 — Irish (Watsonium)

Source: UKLO 2016 — https://www.uklo.org/wp-content/uploads/2022/05/2016_3.-Watsonium.pdf

## Context

Irish (Gaeilge) includes foreign loan words based on Latin, Greek or English. The Irish spelling follows pronunciation rather than original spelling. The acute accent (é) indicates a long vowel.

| English | Irish |
|---------|-------|
| Beryllium | Beiriliam |
| Carbon | Carbón |
| Chlorine | Clóirín |
| Cobalt | Cóbalt |
| Helium | Héiliam |
| Magnesium | Maignéisiam |
| Polonium | Polóiniam |
| Promethium | Próiméitiam |
| Thulium | Túiliam |
