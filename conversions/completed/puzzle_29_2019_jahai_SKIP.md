# Puzzle 29 — Jahai [SKIP]

Source: UKLO 2019 — https://www.uklo.org/wp-content/uploads/2022/05/2019_3-Jahai.pdf

## Reason for skipping

This puzzle cannot be converted to a standard Match-Up format because it has many-to-one mappings: the same Jahai smell verb can match multiple items (e.g., "cnges" maps to both "cooked food" and "eye-watering smoke"; "harim" maps to "fragrant flowers", "perfume", "mushrooms", and "flower-scented soap"). A Match-Up puzzle requires one-to-one pairings between the two lists, which this puzzle's structure does not support.
