# Puzzle 60 — Beja

Source: UKLO 2013 R2 — https://www.uklo.org/wp-content/uploads/2022/09/2013r2.3-Beja.pdf

## Context

'Beja' is the Arabic name for the language which calls itself 'ti bedawye', the unwritten language of a group of mainly nomadic tribes in the north-east corner of Sudan (between the Nile and the Red Sea). It is classified as an 'Afro-Asiatic' language, distantly related to Arabic, Hebrew and Ancient Egyptian. In the following examples, ' stands for a glottal stop.

| Beja | English |
|------|---------|
| ilaga diwiini | The male calf is sleeping. |
| doobaab rhitni | She sees a bridegroom. |
| gwibu | It is a mouse. |
| oomeek kiike | He is not the donkey. |
| tuukaam b'ata | The female camel lay down. |
| iragad winu | The leg is big. |
| tilaga wint kitte | The female calf is not big. |
| uutak tim'ari tamya | The man ate the food. |
| yoob tidbil | She collected an ox. |
| oofaar rhita | She saw the flower. |
| tidooba kadiwta | The bride is not sleeping. |
| uumeek b'iini | The donkey is lying down. |
| uuyaas ookaam danbiil | The dog is collecting the camel. |
| hataay tamaabu | He has eaten a horse. |
| ooyoo diblaab kiike | He has not collected the ox. |
| kil'oob kiidbil | He is not collecting a shell. |
| m'ariit tamtiniit kitte | She cannot eat food. |
| ootak kanriifu | He can meet the man. |
| yam kitdibil | She is not collecting water. |
