# Puzzle 66 — Luiseno

Source: UKLO 2012 R2 — https://www.uklo.org/wp-content/uploads/2022/09/2012r2.2-Luiseno.pdf

## Context

Luiseño is a Uto-Aztecan language spoken in southern California with about 30–40 native speakers. It has no standard written form. The International Phonetic Alphabet is used here: [ʔ] = glottal stop, [q] = a back k, [ʃ] = "sh", [j] = "y", [ʂ] = retroflex s, [x] = "ch" as in loch, [ŋ] = "ng" as in singer, [:] = long vowel.

## Notes
- Q2.1 (4 items: Luiseño→English) and Q2.2 (4 items: English→Luiseño) combined into 8 Match-Up pairs.
- The solver must deduce Luiseño word boundaries within continuous phonetic transcription, as well as the roles of ʂu (question particle), qaj (negation), and word order rules.
- Items 2b and 4d both translate as "Does the woman/teacher see/hit..." — the ambiguity in some translations (where SOV and OVS are indistinguishable) is noted in the original solution.
